# Python个人学习笔记
个人学习Python的笔记（主要是给自己看的，顺手丢到github中，还在继续更新）。主要基础资料来源自[黑马程序员](https://yun.itheima.com/course/1008.html)的教程，进阶与补充部分来自于教材和菜鸟教程，读者也可配合学习。笔者之前学C/C++，因此笔记中涉及到C/C++的部分语法比较区别（不影响实际阅读），另外由于个人能力有限且这是一篇极其私人化的笔记，所以可能不是适合初学者的好教程，其中存在很多谬误与不准确，感谢读者指出，欢迎交流。
